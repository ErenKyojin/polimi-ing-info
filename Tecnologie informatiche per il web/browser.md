# browser HTTP
Ha lo scopo di emettere richieste HTTP, ricevere risposte e restituirle all'interno della finestra. Ha un'architettura notevolmente complessa, contiene
- Descrizione dei documenti
	- Interpreter HTML
	- CSS
- Supporto a linguaggio di programmazione nativo
	- javascript
- Meccanismi di estensione
	- visione di contenuti esterno al browser
